# ①課題番号-プロダクト名

駅天気アプリLite

## ②課題内容（どんな作品か）

- 路線図から選択した駅の天気を調べることができます
- 現在〜7日後を表示することができます

## ③DEMO
指定先に提出します

## ④作ったアプリケーション用のIDまたはPasswordがある場合
なし

## ⑤工夫した点・こだわった点

- HeartRails Express(路線API)とopen-meteo(天気API)の２つを組み込んでいます。
- 路線図から駅をクリック→駅名を取得→HeartRails Expressで緯度・経度を取得→open-meteoで天気情報を取得しています。
- 路線図(模式図)はフリー画像に透明なクリックエリアを重ねる方法があるそうなのですが、縮尺変更に強いSVGで描画することにしました。
